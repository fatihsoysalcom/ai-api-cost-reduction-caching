# AI API Cost Reduction Caching

This Python script simulates AI API calls and demonstrates how to significantly reduce costs by implementing an in-memory cache. It avoids redundant API calls for identical prompts, directly lowering expenses without code changes to the core AI logic.

## Language

`python`

## How to Run

Run the script using: python main.py
